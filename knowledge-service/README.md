# knowledge-service

Independent knowledge ingestion, indexing, and retrieval service for Ops Factory.

Current scope:

- Spring Boot module skeleton
- Business-facing configuration model
- SQLite schema bootstrap
- REST API skeletons for source, document, chunk, retrieval, profile, job, stats, and capability

Planned next steps:

- SQLite repositories
- Tika conversion pipeline
- chunking and metadata extraction
- embedding client integration
- Lucene indexing and retrieval
